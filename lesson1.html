🎮 Ôn Tập Kiến Thức GameMaker Basic - Buổi 1

1) 🛠️ Các Event Cơ Bản

Event là những sự kiện xảy ra trong game. Hãy xem sơ đồ dưới đây để hiểu rõ hơn:

flowchart TD A["🎮 Game bắt đầu"] --> B["⚙️ Create Event
Khởi tạo đối tượng"] B --> C["🔄 Step Event
Chạy mỗi frame"] C --> D{"❓ Có va chạm?"} D -->|Có| E["💥 Collision Event
Xử lý va chạm"] D -->|Không| F{"❓ Có nhấn chuột?"} F -->|Có| G["🖱️ Mouse Event
Xử lý chuột"] F -->|Không| H{"❓ Alarm về 0?"} H -->|Có| I["⏰ Alarm Event
Thực hiện hành động"] H -->|Không| C E --> C G --> C I --> C
  • Create: Chạy 1 lần khi đối tượng được tạo - đặt vị trí, tốc độ, biến
  • Step: Chạy liên tục mỗi frame (60 lần/giây) - logic chính của game
  • Alarm: Bộ đếm ngược, khi về 0 thì chạy lệnh - tạo kẻ địch, đặt thời gian
  • Collision: Khi 2 đối tượng chạm nhau - xử lý va chạm, tăng điểm
  • Mouse: Khi nhấn chuột - bắn súng, chọn menu

📌 Nếu sơ đồ không hiện, kiểm tra mạng (cần tải Mermaid) hoặc thử trình duyệt khác.

2) ⚙️ Các Action Quan Trọng

Action là những hành động mà đối tượng có thể thực hiện. Hãy xem sơ đồ phân loại:

flowchart TD A["🎯 Actions"] --> B["👾 Tạo & Hủy"] A --> C["⏰ Thời gian"] A --> D["🎨 Hiển thị"] A --> E["🎵 Âm thanh"] A --> F["🎮 Điều khiển"] B --> B1["➕ Create instance
Tạo đối tượng mới"] B --> B2["🗑️ Destroy instance
Xóa đối tượng"] C --> C1["⏰ Set alarm
Đặt đồng hồ"] C --> C2["📊 Variable
Tạo biến"] D --> D1["🎨 Set sprite
Thay hình ảnh"] D --> D2["📝 Draw value
Vẽ text/số"] E --> E1["🔊 Play audio
Phát âm thanh"] E --> E2["🔇 Stop audio
Dừng âm thanh"] F --> F1["🧭 Set direction
Đặt hướng"] F --> F2["🏃 Set speed
Đặt tốc độ"] F --> F3["🎯 Set point direction
Hướng theo điểm"]

📋 Chi tiết các Action:

  • Create instance: Tạo đối tượng mới (kẻ địch, đạn, power-up)
  • set_alarm: Đặt đồng hồ đếm ngược (30 step = 0.5 giây)
  • variable: Tạo hoặc thay đổi giá trị biến (điểm số, máu)
  • destroy_instance:
    • Self: Hủy chính đối tượng đang chạy
    • Other: Hủy đối tượng va chạm (trong Collision)
    • Objects: Hủy tất cả instance của một loại object
  • set_sprite: Thay đổi hình ảnh của đối tượng
  • draw_value: Vẽ số hoặc text lên màn hình
  • set_direction & set_speed: Điều khiển di chuyển
  • play_audio & stop_audio: Phát và dừng âm thanh

📌 Nếu sơ đồ không hiện, kiểm tra mạng (cần tải Mermaid) hoặc thử trình duyệt khác.

3) 🕹️ Lập Trình Di Chuyển Nhân Vật

Ý tưởng: Nhấn phím để nhân vật di chuyển, thả phím thì dừng lại.

flowchart TD A["🔄 Mỗi khung hình"] --> B{"❓ Nhấn phím W/A/S/D?"} B -->|W| C["⬆ Đi lên: vspeed = -spd"] B -->|S| D["⬇ Đi xuống: vspeed = spd"] B -->|A| E["⬅ Đi trái: hspeed = -spd"] B -->|D| F["➡ Đi phải: hspeed = spd"] B -->|Không| G["🛑 Đứng yên: hspeed = 0, vspeed = 0"]

📌 Nếu sơ đồ không hiện, kiểm tra mạng (cần tải Mermaid) hoặc thử trình duyệt khác.

4) ⏰ Lập Trình Spawner

Ý tưởng: Tạo kẻ địch định kỳ bằng alarm.

flowchart TD A["⚙️ Tạo spawner"] --> B["⏰ Đặt alarm[0] = 30"] B --> C["⏰ Alarm 0 chạy"] C --> D["👾 Tạo o_enemy"] D --> B

🔧 Cách hoạt động:

  • Create Event: alarm[0] = 120; (2 giây)
  • Alarm 0 Event: Tạo enemy và đặt lại alarm
  • Lặp lại: Cứ mỗi 2 giây tạo 1 enemy mới

📌 Nếu sơ đồ không hiện, kiểm tra mạng (cần tải Mermaid) hoặc thử trình duyệt khác.

5) 📍 Trục Tọa Độ Trong GameMaker

GameMaker sử dụng hệ tọa độ 2D với trục X (ngang) và Y (dọc). Hãy xem sơ đồ:

flowchart TD A["🎯 Hệ tọa độ GameMaker"] --> B["📏 Trục X - Horizontal"] A --> C["📏 Trục Y - Vertical"] B --> B1["➡️ hspeed > 0
Di chuyển sang phải"] B --> B2["⬅️ hspeed < 0
Di chuyển sang trái"] B --> B3["🛑 hspeed = 0
Đứng yên ngang"] C --> C1["⬇️ vspeed > 0
Di chuyển xuống dưới"] C --> C2["⬆️ vspeed < 0
Di chuyển lên trên"] C --> C3["🛑 vspeed = 0
Đứng yên dọc"] D["📍 Gốc tọa độ (0,0)"] --> E["Góc trên trái màn hình"] D --> F["Trục X tăng sang phải"] D --> G["Trục Y tăng xuống dưới"]

🎮 Ví dụ thực tế:

  • Nhân vật ở (100, 200): Cách góc trái 100px, cách góc trên 200px
  • Di chuyển sang phải: hspeed = 5 (tăng x mỗi frame)
  • Di chuyển lên trên: vspeed = -3 (giảm y mỗi frame)
  • Dừng lại: hspeed = 0; vspeed = 0

📌 Nếu sơ đồ không hiện, kiểm tra mạng (cần tải Mermaid) hoặc thử trình duyệt khác.

6) 🌄 Parallax Trong GMS2

Parallax tạo chiều sâu 3D bằng cách di chuyển nền ở tốc độ khác nhau. Hãy xem sơ đồ:

flowchart TD A["🌄 Parallax Effect"] --> B["🏔️ Nền xa - Chậm"] A --> C["🌳 Nền giữa - Trung bình"] A --> D["🌿 Nền gần - Nhanh"] A --> E["🙂 Nhân vật - Tốc độ bình thường"] B --> B1["Speed = 0.5
Di chuyển chậm nhất"] C --> C1["Speed = 1.0
Di chuyển bình thường"] D --> D1["Speed = 2.0
Di chuyển nhanh hơn"] E --> E1["Speed = 1.0
Tốc độ nhân vật"] F["⚙️ Cài đặt Background"] --> G["✅ Stretch
Điều chỉnh kích thước"] F --> H["✅ Horizontal Tile
Lặp lại theo chiều ngang"] F --> I["✅ Speed
Tốc độ di chuyển"]

🎨 Cách tạo Parallax:

  • Stretch: Điều chỉnh kích thước nền (1920×1080 → 1366×768)
  • Horizontal Tile: Nền lặp lại theo chiều ngang tạo hiệu ứng vô tận
  • Speed: Tốc độ di chuyển nền (0.5 = chậm, 2.0 = nhanh)
  • Lớp nền: Nền xa có speed thấp, nền gần có speed cao

📌 Nếu sơ đồ không hiện, kiểm tra mạng (cần tải Mermaid) hoặc thử trình duyệt khác.

7) 📋 Hướng Dẫn Thực Hành

Hãy làm theo các bước dưới đây để tạo game đầu tiên của bạn:

flowchart TD A["🚀 Bắt đầu tạo game"] --> B["1️⃣ Tạo Object
o_player, o_enemy, o_spawner"] B --> C["2️⃣ Tạo Room
Đặt các object vào room"] C --> D["3️⃣ Lập trình Player
Create: đặt tốc độ = 0"] D --> E["4️⃣ Lập trình di chuyển
Step: đọc phím W/A/S/D"] E --> F["5️⃣ Lập trình Spawner
Alarm: tạo enemy định kỳ"] F --> G["6️⃣ Lập trình Enemy
Step: di chuyển sang trái"] G --> H["7️⃣ Xử lý va chạm
Collision: xóa player & enemy"] H --> I["🎮 Chạy thử game!"]

📝 Chi tiết từng bước:

  • Bước 1: Tạo các Object cần thiết (o_player, o_enemy, o_spawner)
  • Bước 2: Tạo Room và đặt các object vào vị trí phù hợp
  • Bước 3: Trong Create Event của o_player: hspeed = 0; vspeed = 0; spd = 5
  • Bước 4: Trong Step Event: đọc phím và đặt hspeed, vspeed
  • Bước 5: Trong o_spawner: Create Event đặt alarm, Alarm Event tạo enemy
  • Bước 6: Trong Step Event của o_enemy: hspeed = -2
  • Bước 7: Tạo Collision Event giữa o_player và o_enemy

📌 Nếu sơ đồ không hiện, kiểm tra mạng (cần tải Mermaid) hoặc thử trình duyệt khác.

8) 🎯 Tóm tắt buổi học

Trong buổi học này, chúng ta đã học được:

  • ✅ Các Event cơ bản: Create, Step, Alarm, Collision, Mouse
  • ✅ Các Action quan trọng: Create instance, set_alarm, variable, destroy_instance
  • ✅ Lập trình di chuyển nhân vật bằng phím W/A/S/D
  • ✅ Tạo spawner để sinh kẻ địch định kỳ
  • ✅ Hiểu về hệ tọa độ và Parallax trong GameMaker

Bài tập về nhà: Thử tạo một game đơn giản với nhân vật di chuyển và spawner tạo kẻ địch!
